### 按场景选择

| 需求 | 推荐参考 |
|------|----------|
| 学习结构化格式 | `personas/` 目录 |
| 小红书运营 | `xiaohongshu/` 目录 |
| 创作类任务 | `creative/` 目录 |
| 找特定功能 | `gpts-personas/` 目录 |
| 研究系统提示词 | `system-tools/` 目录 |

| 类型 | 适用模式 | 特点 |
|------|----------|------|
| personas | 详细模式 | 多维知识框架、树形交互逻辑 |
| xiaohongshu | 详细模式 | 三层诊断法、人感理论 |
| creative | 简单模式 | 骨骼结构、戒律约束 |
| gpts-personas | 混合模式 | 各种风格、功能导向 |
| system-tools | 系统级 | 工具链集成、专业指令 |
